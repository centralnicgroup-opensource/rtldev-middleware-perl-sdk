NAME

WebService::Hexonet::Connector - Connector library for the insanely fast HEXONET Backend API.

SYNOPSIS

        ###############################
        # How to use this Library?
        ###############################

        # Install our module by
        cpan WebService::Hexonet::Connector
        # or
        cpanm WebService::Hexonet::Connector
        # NOTE: We suggest to use cpanm (App::cpanminus) for several reasons.

Check the Example provided at WebService::Hexonet::Connector::APIClient.

DESCRIPTION

This module is used as namespace.

AVAILABLE SUBMODULES

We've split our functionality into submodules to give this module a better structure.

WebService::Hexonet::Connector::APIClient - API Client functionality.
WebService::Hexonet::Connector::Column - API Response Data handling as "Column".
WebService::Hexonet::Connector::Record - API Response Data handling as "Record".
WebService::Hexonet::Connector::Response - API Response functionality.
WebService::Hexonet::Connector::ResponseParser - API Response Parser functionality.
WebService::Hexonet::Connector::ResponseTemplate - API Response Template functionality.
WebService::Hexonet::Connector::ResponseTemplateManager - API Response Template Manager functionality.
WebService::Hexonet::Connector::SocketConfig - API Communication Configuration functionality.
